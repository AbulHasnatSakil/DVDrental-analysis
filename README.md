# DVDrental analysis

The DVD rental database represents the business processes of a DVD rental store.<br/>
The DVD rental database has many objects including:

  - 15 tables<br/>
  - 1 trigger<br/>
  - 7 views<br/>
  - 8 functions<br/>
  - 1 domain<br/>
  - 13 sequences<br/>
